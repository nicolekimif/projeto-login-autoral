# Projeto Login Autoral

## Descrição

Projeto criado, com auxílio do professor [Leonardo Rocha](https://github.com/leonardossrocha) na aula de programação Front-End, na Unicesumar, campus de Londrina. Durante o desenvolvimento do projeto, foram utilizados os conceitos básicos de HTML e CSS para a criação de uma página de login autoral, com foco na estruturação da interface e na estilização dos elementos da página. Também foram aplicados conceitos de Git e GitHub para versionamento e publicação do projeto.

## Tecnologias utilizadas

- HTML5
- CSS3
- Visual Studio Code
- Git

## Funcionalidades

- Interface de login simples
- Campos para usuário e senha
- Botão de acesso
- Estilização personalizada com CSS

## Objetivo do projeto

Este projeto foi desenvolvido com fins educacionais, visando aprimorar conhecimentos em desenvolvimento web front-end, especialmente na utilização de HTML para estruturação e CSS para estilização de páginas.
